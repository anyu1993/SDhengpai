# 山东恒派智能科技有限公司 网站规格书

## 1. Concept & Vision

打造一个具有强烈工业科技感的公司网站，展现环保水处理设备制造商的专业形象。整体氛围：深邃的深蓝色背景配合流动的光线条，传递"科技、环保、工业力量"的感觉，让访客感受到企业技术实力和产品可靠性。

## 2. Design Language

### 色彩系统
- 主色（Primary）: `#0077FF` - 科技蓝
- 次色（Secondary）: `#00C7BE` - 青色点缀
- 强调色（Accent）: `#5856D6` - 紫色渐变
- 背景深色: `#0a0f1a` - 深邃夜空
- 背景次深: `#0d1526` - 卡片背景
- 文字主色: `#FFFFFF`
- 文字次色: `#94A5C7` - 柔和蓝灰
- 发光效果: `rgba(0, 119, 255, 0.3)` - 蓝色光晕

### 字体
- 标题: "Noto Sans SC", "Source Han Sans CN", sans-serif (700 weight)
- 正文: "Noto Sans SC", sans-serif (400 weight)
- 英文/数字: "Inter", sans-serif

### 空间系统
- 基础单位: 8px
- 段落间距: 24px
- 区块间距: 80px (移动端 48px)
- 最大内容宽度: 1200px

### 动效哲学
- 粒子背景：缓慢漂浮的光点，营造科技氛围
- 滚动视差：Banner区域随滚动轻微位移
- 悬停反馈：150ms ease-out 过渡，卡片上浮+发光边框
- 页面切换：淡入淡出

### 视觉元素
- 图标: Lucide Icons (线性风格，2px描边)
- 产品图片: 示意图（蓝色发光边框+深色背景）
- 装饰: 流动光线条、渐变圆形、网格背景

## 3. Layout & Structure

### 页面结构
```
┌─────────────────────────────────────┐
│ Navigation (固定顶部，毛玻璃效果)    │
├─────────────────────────────────────┤
│ Hero Banner (全屏，粒子背景)         │
│ 标题 + 描述 + CTA按钮                │
├─────────────────────────────────────┤
│ Products Section (产品展示网格)      │
│ 4产品卡片 + 悬浮详情                  │
├─────────────────────────────────────┤
│ About Section (公司介绍)             │
│ 优势 + 数据统计                      │
├─────────────────────────────────────┤
│ Contact Section (联系信息)           │
│ 地图占位 + 联系卡片                   │
├─────────────────────────────────────┤
│ Footer (深色，版权信息)               │
└─────────────────────────────────────┘
```

### 响应式断点
- Desktop: > 1024px (4列产品网格)
- Tablet: 768px - 1024px (2列网格)
- Mobile: < 768px (单列，卡片全宽)

## 4. Features & Interactions

### Navigation
- 固定顶部，滚动后添加背景模糊
-  Logo + 导航链接 + 联系电话
-  移动端：汉堡菜单，侧边滑出

### Hero Banner
- 全屏高度，粒子动画背景
- 主标题 + 副标题 + "联系我们" 按钮
- 向下滚动箭头动画

### 产品卡片
- 深色卡片 + 发光边框（悬停时）
- 产品图标/示意图 + 名称 + 简介
- 悬停：上浮8px + 边框发光增强

### 联系表单
- 姓名、电话、留言三字段
- 提交按钮带加载状态
- 基础验证：必填项 + 电话格式

## 5. Component Inventory

### Navbar
- 默认：透明背景
- 滚动后：`bg-[#0a0f1a]/80 backdrop-blur-lg`
- 链接悬停：文字变为主色蓝色

### Product Card
- 默认：深色背景 `#0d1526`，细边框 `rgba(0,119,255,0.2)`
- 悬停：边框变亮 `rgba(0,119,255,0.6)`，上浮 `translateY(-8px)`
- 内含：图标区 + 标题 + 描述

### CTA Button
- 默认：填充主色，文字白色
- 悬停：亮度提升，轻微放大
- 点击：缩小反馈

### Section Title
- 英文小标题（淡蓝色）+ 中文大标题
- 下方蓝色短横线装饰

## 6. Pages

### index.html (首页)
- Hero Banner
- 产品展示（4款产品入口）
- 公司简介
- 联系方式
- Footer

### products.html (产品中心)
- 页面Banner
- 产品详情网格（所有产品）
- 应用领域

### about.html (关于我们)
- 公司介绍
- 核心优势（数据展示）
- 发展历程

### contact.html (联系我们)
- 联系信息卡片
- 简易联系表单
- 公司地址

## 7. Technical Approach

- **技术栈**: 纯HTML5 + TailwindCSS (CDN) + Vanilla JavaScript
- **图标**: Lucide Icons CDN
- **字体**: Google Fonts (Noto Sans SC, Inter)
- **动画**: CSS动画 + 轻量Canvas粒子
- **无框架依赖**: 便于部署和维护
